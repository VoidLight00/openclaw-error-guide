Mac용 OpenClaw 설치 가이드

macOS 사용자를 위한 단계별 설치 안내

!
사전 준비
  • Anthropic (Claude) 계정 — Pro $20/월 또는 Max $100/월 구독 필요
  • 터미널 열기: Cmd + Space → "터미널" 입력 → Enter
설치 순서: Node.js → OpenClaw. OpenClaw은 Node.js 22+ 없이는 설치 자체가 안 됩니다.
1
Node.js 22+ 설치

OpenClaw은 Node.js 22.12.0 이상에서만 동작합니다.

이미 설치되어 있는지 확인

Terminal 
node --version

v22.12.0 이상이 나오면 → Step 2로 건너뛰세요!

방법 1: 공식 설치 프로그램 (초보자 추천)

  • nodejs.org 접속 → v22.x LTS 다운로드 (macOS Installer .pkg)
  • 다운로드된 .pkg 파일 실행 → 설치 마법사 따라 진행

방법 2: Homebrew

Terminal 
brew install node@22

설치 확인

Terminal (새로 열기) 
node --version   # v22.x.x 이상
npm --version    # 10.x.x 이상
2
OpenClaw 설치

오픈소스 AI 에이전트 플랫폼. 다양한 모델, 스킬, 메시징 앱 연동을 지원합니다.

Node.js 버전 재확인: node --version → v22.12.0 이상이 아니면 Step 1로 돌아가세요!

방법 1: 설치 스크립트 (추천)

Terminal 
curl -fsSL https://openclaw.ai/install.sh | bash

방법 2: npm 직접 설치

Terminal 
npm install -g openclaw@latest
텔레그램으로 AI와 대화하고 싶다면?
온보딩 ④단계에서 바로 연결할 수 있도록, 텔레그램 봇 토큰을 미리 준비해두세요!
텔레그램 봇 만들기 가이드 →

텔레그램 외 다른 채널(Discord, WhatsApp 등)도 온보딩에서 설정 가능합니다.

3
OpenClaw 온보딩 (초기 설정)
Terminal 
openclaw onboard --install-daemon

온보딩 5단계:

  • ① AI 모델/인증 선택 — Anthropic API 키 또는 Claude Code 로그인 정보
  • ② 워크스페이스 생성 — 기본: ~/.openclaw/workspace
  • ③ 게이트웨이 설정 — 기본 포트 18789
  • ④ 채널 연결 — 텔레그램 봇 토큰 입력 (미리 준비했다면 바로 연결!)
  • ⑤ 데몬 설치 — 백그라운드 자동 실행
초보자는 QuickStart 모드를 선택하세요!
설치 확인
Terminal 
openclaw gateway status   # "running"이 보이면 성공!
openclaw doctor           # 전체 진단
openclaw dashboard        # 웹 대시보드 열기
"running"이 보이면 설치 완료! 다음 단계로 텔레그램 연동을 진행하세요.
자주 발생하는 문제
증상 해결 방법
command not found 터미널을 닫고 새로 열어서 다시 시도 (PATH 반영 문제)
permission denied 명령어 앞에 sudo 추가: sudo npm install -g openclaw@latest
Homebrew로 설치한 Node가 안 잡힘 brew link node@22 실행 후 터미널 재시작
어떤 에러인지 모르겠음 아래 3줄 실행 후 결과를 도우미에게 보여주세요
진단 명령어 
node --version
openclaw --version
openclaw doctor
