AI 코딩 에이전트로 OpenClaw 설치하기

Claude Code, Codex, Antigravity 중 하나만 있으면
프롬프트 복붙으로 자동 설치 완료

왜 코딩 에이전트로 설치하나요?
  • 터미널 명령어를 몰라도 AI가 알아서 실행합니다
  • OS별로 다른 오류도 AI가 즉시 해결합니다
  • Node.js 설치부터 온보딩까지 한 번에 끝납니다
사전 준비물: Anthropic (Claude) Pro/Max 구독 + 아래 3개 중 하나 설치 완료 상태여야 합니다.
?
어떤 에이전트를 쓰면 되나요?
에이전트 제공사 필요 구독 Mac Windows
Claude Code Anthropic Claude Max ($100/월) O O
Codex OpenAI ChatGPT Pro ($200/월) O O
Antigravity Antigravity 별도 가입 O O

3개 모두 터미널 명령어 실행이 가능해서 OpenClaw 자동 설치를 지원합니다.

1
코딩 에이전트 설치 확인

이미 설치되어 있다면 Step 2로 건너뛰세요.

Claude Code Anthropic

Terminal 
npm install -g @anthropic-ai/claude-code

설치 후 claude 명령어로 실행. 처음 실행 시 Anthropic 로그인 필요.

PowerShell (관리자) 
npm install -g @anthropic-ai/claude-code

Windows에서는 PowerShell을 관리자 권한으로 열어서 실행하세요.

Codex (OpenAI) OpenAI

Terminal 
npm install -g @openai/codex

설치 후 codex 명령어로 실행. OpenAI API 키 또는 로그인 필요.

PowerShell (관리자) 
npm install -g @openai/codex

Windows에서는 PowerShell을 관리자 권한으로 열어서 실행하세요.

Antigravity Antigravity

Terminal 
curl -fsSL https://antigravity.dev/install.sh | bash

설치 후 antigravity 또는 ag 명령어로 실행.

PowerShell (관리자) 
irm https://antigravity.dev/install.ps1 | iex

Windows에서는 PowerShell을 관리자 권한으로 열어서 실행하세요.

2
프롬프트 복붙 → OpenClaw 자동 설치

아래 프롬프트를 코딩 에이전트에 그대로 붙여넣으세요. AI가 알아서 설치합니다.

3개 에이전트 모두 같은 프롬프트를 사용합니다. 아무 에이전트에나 붙여넣으면 됩니다.

OpenClaw 설치 프롬프트

OpenClaw을 설치해줘. 순서대로 진행해: 1. Node.js 버전 확인 (node --version). 22.12.0 이상이 아니면 설치해줘. - Mac: brew install node@22 (Homebrew 없으면 먼저 설치) - Windows: winget install OpenJS.NodeJS.LTS 또는 공식 사이트에서 설치 2. OpenClaw 설치: npm install -g openclaw@latest 3. 설치 확인: openclaw --version 4. 온보딩 실행: openclaw onboard --install-daemon (대화형 설정이 나오면 안내해줘) 5. 게이트웨이 상태 확인: openclaw gateway status 각 단계마다 결과를 보여주고, 오류가 나면 해결해줘.

사용 방법

Claude Code에서 사용

Terminal 
claude

실행 후 위 프롬프트를 붙여넣기 → Enter

Codex에서 사용

Terminal 
codex

실행 후 위 프롬프트를 붙여넣기 → Enter. full-auto 모드 권장: codex --approval-mode full-auto

Antigravity에서 사용

Terminal 
ag

실행 후 위 프롬프트를 붙여넣기 → Enter

3
설치 완료 확인

AI가 설치를 마치면 아래 명령어로 직접 확인하세요.

Terminal 
openclaw gateway status   # "running"이 보이면 성공!
openclaw doctor           # 전체 진단
"running"이 보이면 설치 완료! 텔레그램 연동은 여기를 참고하세요.
코딩 에이전트가 없다면?
  • Mac 사용자: Mac 수동 설치 가이드를 따라하세요
  • Windows 사용자: Windows 수동 설치 가이드를 따라하세요
  • 둘 다 어렵다면: 강의 당일 현장에서 도와드립니다. 미리 설치하지 않아도 괜찮습니다!
설치 중 문제가 있으면 VoidLight 채팅방에서 질문하세요!
