# OpenClaw Error Guide Skill

Search and retrieve OpenClaw installation/setup error solutions.

## Usage
When a user encounters an OpenClaw error, search the error database:

```bash
node {{SKILL_DIR}}/search.js "error keyword"
```

## Examples
- User says "EACCES permission denied" → search for EACCES
- User says "openclaw command not found" → search for "command not found"
- User says "Telegram bot not responding" → search for "Telegram"
